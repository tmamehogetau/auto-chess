# MVP仕様書（統合版） v0.1.2

> 対象: PC / 4人フレンドのルームマッチ
>
> ゲーム種別: **1:3 非対称レイド型オートチェス**（ボス1人 vs レイド3人）
>
> 世界観: **東方Project二次創作**（異変の再演 / 模造異変）
>
> 本書は、当初のMVP仕様 v0.1 を土台に、ここまで固まった仕様をすべて追記・統合した単一のMarkdown仕様書である。

---

## 0. MVPの目的（検証項目）

1. **1:3 非対称**がメイン体験として成立するか
2. **フェーズHP（削り切り目標）→経済加速**が「攻める理由」になるか
3. **最終ラウンド（R12）の純耐久**がクライマックスとして盛り上がるか
4. **共有プール（取り合い）**がゲーム性を増やしつつ、体験を壊さないか
5. **ボス基本有利（目標勝率60%）**が、楽しく納得感のある体験として成立するか
6. **盤面共有 + 主人公常駐**が窮屈すぎず、構築の楽しさを維持できるか
7. **東方らしい関係性 / 異変構造**がシステムに翻訳され、世界観の説得力があるか

---

## 1. 対象プレイ形態

- **PC**
- **4人フレンドのルームマッチ**前提
- 野良マッチングはMVP範囲外
- 永続メタ進行（アンロック、コレクション等）はMVP範囲外
- 戦闘は**フルオート**
- 情報は**可能な限り開示**する方針

## 2. 世界観 / 解釈方針

### 2.1 世界観の基本解釈

- 本作は**東方Project二次創作ゲーム**である
- ボスは「異変を起こした本人」そのものではなく、**異変を模した存在 / 再演された異変 / 偽の異変首謀者**として扱う
- これにより、原作上関係のあるキャラクターがレイド側（解決側）としてボスに対抗する構図を成立させる

### 2.2 ボス側の立ち位置

- 各作品の異変を起こした**首謀者サイド**をボス候補とする
- ボスは単なる強キャラではなく、**異変そのものを展開する存在**
- ボスの強さは、キャラごとに以下の比率が異なってよい
  - 本体性能
  - スペル / ギミック
  - 環境支配

### 2.3 レイド側の立ち位置

- レイド側は**異変解決側の主人公 / 自機相当キャラ**
- 原作作品をまたいで、後年に自機を務めるキャラ（例: 早苗）を含めてよい
- 関係性のあるキャラクターがボスを止めに来る構図を重視する

### 2.4 世界観をMVPで落とし込む方針

MVPで必ず入れるべきもの:
- 異変構造（首謀者 vs 解決側）
- スペルカードの「宣言される勝負」感
- キャラごとの能力テーマ
- ボスごとの作品 / 陣営らしさ

MVPで後回しにするもの:
- 会話演出
- 詳細なストーリー
- ボイス
- 原作弾幕の厳密再現

## 3. ルーム設定

### 3.1 ボス決定方法（ホスト選択）

- **ランダム**: 開始時にランダムで1人がボス
- **指名**: 開始前にホストがボスを指名

> MVPでは拒否権などの社交保険は未導入。

## 4. 勝利条件

### 4.1 レイド側（3人）

- **最終ラウンド（R12）終了時点で、1人でも生存していれば勝利**

### 4.2 ボス側（1人）

- レイド側3人を**全員脱落**させれば即勝利
- または**支配カウント 5** 到達で即勝利（R12では停止）

---

## 5. マッチ構造（ラウンド）

- ラウンド数: **最大12**
- フェーズHP有効: **R1〜R11**
- **R12はフェーズHPなし（純耐久）**

### 5.1 各ラウンド進行（固定時間）

- **ラウンド開始時**: ボスがスペルを宣言（該当ラウンドのみ）
- **準備フェーズ: 25秒（固定）**
- **戦闘フェーズ: 30秒（固定）**
- 結果表示・報酬: 短時間表示

> MVPではReady即開始は入れない。検証再現性を優先する。

## 6. 盤面

### 6.1 ボス側盤面

- **7×4**

### 6.2 レイド側盤面

- **6×4**
- レイド3人は**同一盤面を共有**する
- 盤面共有前提で、各プレイヤーは協力して配置を最適化する

## 7. 出撃仕様（重要変更）

### 7.1 出撃上限

- 開始レベル: **Lv3（出撃3）**
- 最大レベル: **Lv7（出撃7）**

### 7.2 主人公枠（レイド側）

- 各レイドプレイヤーは、マッチ開始時に**主人公キャラを1体選択**する
- 主人公は**常時盤面に存在する**
- 主人公は**出撃枠を消費しない**
- よってレイド側は、共有盤面に
  - **主人公3体（固定）**
  - **自由出撃枠 最大7体**
  を同時に展開できる

### 7.3 主人公の扱い

- 売却不可
- 常時存在
- MVPでは星上げなし
- 主人公は「方向性提示役」であり、編成の核だが、通常ユニット構築も重要である

## 8. 経済

### 8.1 基本収入

- 毎ラウンド: **+5G**
- 勝敗ボーナス（簡易）: **勝ち +1G / 負け +1G（同額）**

### 8.2 リロール

- リロール: **2G**
- 同一ラウンド内のリロールは**3回目以降 +1G**（2 → 3 → 4...）

### 8.3 経済思想

- ボスは**経済で大きく有利にしない**
- ボスの強さは主に
  - システム
  - 盤面補正
  - スペル
  - 専用ショップ
 で作る

## 9. ショップ / 共有プール

### 9.1 通常ショップ

- 全プレイヤー共通の通常ショップ: **5枠**
- 共通プールから抽選

### 9.2 ベンチ枠数

- **8枠**

### 9.3 共有プール仕様

- 全プレイヤー共通の在庫（プール）から抽選
- **購入**でプールから減る
- **売却**でプールへ戻る
- **脱落時**: 脱落者の盤面 + ベンチのユニットを**全てプールへ戻す**

### 9.4 プール枚数（各ユニット在庫）

- 1G: **18枚**
- 2G: **14枚**
- 3G: **11枚**
- 4G: **8枚**
- 5G: **6枚**

### 9.5 残り枚数UI

- MVPでも最低限、**枯渇警告**を出す
- 可能なら残数表示

## 11. 合成（星上げ）

### 11.1 合成ルール

- 同名ユニットが**3体**揃うと**自動で2★に合成**
- 合成は**準備フェーズ中のみ**
- 3★はMVPではなし

### 11.2 2★強化（倍率のみ・挙動変化なし）

- HP × **1.8**
- 攻撃力 × **1.7**
- スキル係数 × **1.35**
- CC時間 × **1.15**

### 11.3 売却価格

- 1★: コストと同額
- 2★: **コスト×2**

### 11.4 売却時のプール戻し枚数

- 1★売却: **1枚戻る**
- 2★売却: **3枚戻る**

## 12. 戦闘基本（MVP単純化）

### 12.1 ターゲット

- 通常攻撃ターゲット: **最も近い敵**

### 12.2 射程

- 近接: Range **1**
- 遠隔: Range **3**
- 術師 / 狙撃: Range **4**（該当ユニットのみ）

### 12.3 マナ

- 全ユニット: **開始0 / 最大100**
- 100到達でスキル発動

### 12.4 フルオート戦闘の原則

- 戦闘中の手動操作なし
- 戦闘中の予兆は、行動変更できない限り原則としてゲーム的意味を持たせない
- 重要情報は**準備フェーズ前 / 準備フェーズ中**に開示する

## 13. 紋章マス

### 13.1 種類（3種）

- 猛攻（赤）: 与ダメ **+15%**
- 守護（青）: 被ダメ **-15%**
- 疾風（緑）: 初回スキル発動が**20%早い**

### 13.2 使用数

- レイド側: 毎ラウンド **1個有効**
- ボス側: 毎ラウンド **2個有効**
- フェーズ失敗の次ラウンド: ボス側 **+1個（合計3）**

## 14. コアシステム: フェーズHP（R1〜R11）

### 14.1 判定

- レイド側全体がボスに与えた**総ダメージ**を集計
- 総ダメージがフェーズHP以上なら**成功**
- 未満なら**失敗**
- フェーズHPは**ラウンド持ち越しなし**

### 14.2 フェーズHP（仮カーブ）

- R1: 600
- R2: 750
- R3: 900
- R4: 1050
- R5: 1250
- R6: 1450
- R7: 1650
- R8: 1850
- R9: 2100
- R10: 2400
- R11: 2700
- R12: なし

### 14.3 成功報酬（経済加速）

レイド側全員:
- **+3G**
- 次ラウンド、各レイド側ショップに**噂勢力ユニット最低1体確定出現**
  - 共有プール枯渇時は例外生成可

個人ボーナス（最後の一撃者のみ、次ラウンド開始時に1つ選択）:
- 初期マナ +20（次戦のみ）
- 与ダメ +10%（次戦のみ）
- 無料リロール×1（次ショップのみ）

### 14.4 失敗時（ボス優勢）

- **支配カウント +1**
- 次ラウンドのみ、ボスは**紋章 +1個**

### 14.5 支配カウント

- 初期0、失敗ごとに+1
- **5到達でボス勝利**
- R12では停止

## 15. スペルカードシステム（ラウンドギミック）

### 15.1 基本思想

- フェーズHPは、東方原作の**スペルカードシステム**の再解釈である
- ボスは**ラウンド開始時にスペルを宣言**する
- 宣言後に準備フェーズが始まる
- スペルは**そのラウンドのみ有効**
- スペル効果は原則として**単発**（持続型で次ラウンドに持ち越さない）

### 15.2 スペル運用構造

- 各セットでボスに**3枚のスペル**が配られる
- ボスはそれを複数ラウンドに渡って**使い切る**
- スペルは事前に全公開し、使用済み状態も表示する

### 15.3 強度段階

- R1〜R4: 1巡目（通常版）
- R5〜R8: 2巡目（強化版）
- R9〜R11: 3巡目（さらに強化）
- R12: ラストスペル相当の特別演出

### 15.4 スペル効果の原則

- 1行で説明できる
- 明確な対策がある
- 数値暴力だけではなく、**方向性を歪める**
- ボスを有利にするが、理不尽すぎない

---

## 16. 攻撃存在（スペル専用フィールドギミック）

### 16.1 基本思想

- スペルによって、**攻撃だけを発生させる存在**を盤面に出してよい
- これは通常ユニットではなく、**フィールドギミック**として扱う

### 16.2 ルール

- ターゲット不可
- バフ対象外
- CC対象外
- フェーズHP対象外
- 破壊不可
- そのラウンド限定
- 同時最大2つ程度まで

### 16.3 ダメージ原則

- 基本は**固定値**で扱う
- ただし、ボス個別スペルの主力技（例: レミリアのレーザー）は、キャラ性に応じてATK依存でもよい

---

## 17. スキル当たり判定・対象選択（MVP固定）

### 17.1 距離・範囲

- 距離は**チェビシェフ距離**: `max(|dx|, |dy|)`
- `radiusTiles = N` は `distance <= N` の正方形範囲

### 17.2 同点解決（決定論）

1. HP%が低い
2. マナが高い
3. 距離が近い
4. entityIdが小さい

### 17.3 スキル共通

- 発動時に対象不在なら再ターゲットして発動
- CCはMVPでは**上書き（残り時間最大）**で実装可

### 17.4 スキルタイプ別の挙動

- `DAMAGE_SINGLE`: 通常攻撃ターゲット
- `DAMAGE_AOE`: 通常攻撃ターゲット座標中心
- 連鎖: `bounces` は命中する総数
- `BUFF_AS`: 推定DPSが高い味方優先
- `CC_SLOW_AOE`: 移動速度のみ低下（必要に応じて個別例外で攻速低下スキルは別定義）
- `CC_ROOT_AOE`: 移動不可（攻撃 / スキル可）
- `CC_SILENCE_SINGLE`: スキル発動不可（マナ蓄積可）
- `TAUNT_AOE`: 挑発者を優先ターゲット
- `SUMMON`: 隣接タイルに召喚（空きがなければ近い空き） / 1ユニットあたり召喚上限2体

## 18. レイド主人公（正式確定分）

### 18.1 主人公共通方針

- マッチ開始時に1人1体選択
- 出撃枠を消費しない
- 役割分業を成立させるため、各主人公は明確な役割を持つ
- 難易度差は数値差ではなく、**最適解を探す難しさ**で表現する

### 18.2 ⛩ 博麗霊夢

**役割:** 安定 / 弱体 / 基準枠

**スキル:「博麗結界」**
- 最も敵が密集している地点を中心
- 半径2マスの円形範囲
- 4秒間、範囲内の敵に:
  - 与ダメージ -20%
  - 被ダメージ +10%

**難易度:** 低

### 18.3 🗡 十六夜咲夜（レイド版）

**役割:** 制御 / 範囲減速 / 上級向け

**スキル:「スロウ・ワールド」**

- 半径2マスの円形範囲
- 3秒間、範囲内の敵に:
  - 移動速度 -60%
  - 攻撃速度 -30%

**難易度:** 高

### 18.4 🧙‍♀️ 霧雨魔理沙

**役割:** 純火力 / フェーズ削り

**スキル:「マスタースパーク」**

### 18.5 🌿 東風谷早苗

**役割:** 全体バフ / チーム底上げ

**スキル:「奇跡『神風の祝福』」**

## 20. ボス第一弾: レミリア（確定仕様）

### 20.1 コンセプト

- 単体怪物
- 条件付きで怪物化
- 専用軍勢で補強
- **CCに弱い**

### 20.2 基礎ステータス（暫定）

- HP: **3200**
- ATK: **280**
- AS: **0.95**
- 物理軽減: **15%**
- 魔法軽減: **10%**

> ボス側の常時盤面補正として、別途HP+10%を与える想定。

### 20.3 パッシブ: 「紅き夜の王」

HP70%以上の間:
- ATK +20%
- 与えたダメージの10%を回復（通常攻撃 / スキル / レーザーすべて対象）
- HPが70%以上に戻れば再発動

制限:
- 回復はフェーズHPとは無関係
- 1回の攻撃イベントごとの回復上限は、必要に応じて最大HPの25%程度で安全装置を入れてよい

### 20.4 明確な弱点

- CC耐性なし
- スタン / 減速 / 行動阻害に弱い
- CC中は吸血できない
- レーザーもCC中は発動不能として扱ってよい

### 20.5 スペルの性格

- 前半は読み合い
- 後半ほど強力になる
- レーザーはATK依存で段階強化

### 20.6 代表スペル

#### 「紅符『スカーレットデスレーザー』」

- 発動時、現在の通常攻撃ターゲット方向へ直線発射
- **ATK依存ダメージ**
- クリティカルなし
- 後半ラウンドほど幅 / 倍率が上がる

段階例:
- R1〜R4: 幅1 / ATK×2.3
- R5〜R8: 幅1 / ATK×2.7
- R9〜R11: 幅2 / ATK×3.0

#### そのほかのスペル枠（性格確定）

- 前列被ダメ増 + 回復量減少（制圧）
- 一定時間AS増 + 被ダメ軽減（暴走）

### 20.7 専用ショップ（紅魔館枠）

- レミリアは専用ショップを持つ
- **常時2枠**
- 専用ショップからのみ紅魔館ユニットが出る
- 通常ショップとは別枠で考える
- 常時全部解禁

### 20.8 紅魔館ユニット

#### 紅美鈴

- コスト: 2G
- 役割: 序盤の壁
- コンセプト: 1★で仕事がほぼ完結する門番
- 終盤は自然に力不足になる

#### 十六夜咲夜（ボス版 / 従者版）

- コスト: 3G
- 役割: 守護サポート
- コンセプト: レミリアの70%ライン維持を助ける
- レイド版とは役割を明確に分ける

#### パチュリー・ノーレッジ

- コスト: 4G
- 役割: 爆発補助 / 直線支援
- コンセプト: 単騎では怪しいが、レミリアと組むと強い
- 低HP / 低耐久で、守られないと落ちやすい
- スキルで被ダメ増を付与し、それがレミリアのレーザーにも乗る

### 20.9 同時出場制限

- レミリアを除く紅魔館ユニットは、**同時出場2体まで**

### 20.10 紅魔館シナジー: 「紅魔の夜」

発動条件:
- 紅魔館タグ2体出場（レミリア除く）

効果:
- レミリアがHP70%以上の間のみ
  - ATK +10%
  - レーザー +10%
  - 吸血 +5%

方針:
- 情報は常時表示
- ボスは迷ったら強めにする方針に従い、数値は保守的に弱めず採用
- 3体目シナジーは作らない

### 20.11 フランの扱い

- フランドールはEXボス格として、MVPでは原則保留
- 採用する場合は、レミリアと同時出場不可の特別枠にする想定

## 27. 現在の設計要旨（要約）

- レイド3人は**盤面共有**で完全協力
- 主人公は**出撃枠を使わず常駐**

- ボスの強さは主に**構造 / スペル / 専用軍勢**で作る
- レミリアは「条件付き怪物 + 射線爆発 + CC弱点」の基準ボス
- レイド主人公5人は
  - 咲夜（制御）
  - 霊夢（安定）
  - 魔理沙（削り）
  - 早苗（バフ）
  - 妖夢（前衛DPS）
  で役割分業が成立している
- 世界観は「異変の再演 / 模造異変」で整合を取る
